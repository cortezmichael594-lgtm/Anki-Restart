### Anki Restart by Ankicraft

Ever crash-tested your patience waiting for Anki to reboot after a sync hiccup or a slow add-on reload? Alt-tabbing to find the icon, double-clicking, waiting for the splash screen... Quick Restart turns that whole dance into a single keystroke. Quick Restart gives Anki the instant restart it's missing. ⚡

* 🔄 **Restart Anki instantly** with a customizable shortcut — no mouse, no hunting for icons
* ❌ **Close Anki** with its own separate shortcut, just as fast
* 🧠 **Safe by design:** your collection is saved and closed the normal way before restarting, backups included
* ⌨️ **Fully remappable shortcuts**, or turn either one off if you only need the other
* 🛡️ **Built-in guard** against setting both shortcuts to the same key combo
* 🌍 **Interface translated** into dozens of languages, following Anki's own language setting

### HOW TO USE
<hr style="height: 0.5px; border: none; background-color: #e1e4e8;">
1. Install the add-on and restart Anki once.
2. Press **Ctrl+R** anytime to restart Anki, or **Ctrl+W** to close it.
3. Open the config screen to change either shortcut or disable one of them.

### SETTINGS
<hr style="height: 1px; border: none; background-color: #e1e4e8;">
Go to **Tools → Add-ons → Quick Restart → Config** to change the shortcuts or enable/disable restart and close independently.

### COMPATIBILITY
---
Tested on the latest Anki release on Windows; should work the same on macOS and Linux since it only uses standard Qt/Anki APIs, but hasn't been verified there yet.
